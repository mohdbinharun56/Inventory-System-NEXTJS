to added cart implent localStorage which are stored the cart value.
# LocalStorage method
```bash
- SetValue: localStorage.setItem('key','value');
- GetValue: localStorage.getItem('key');
- Remove: localStorage.remove('key);
- Clear: localStorage.clear();
```
